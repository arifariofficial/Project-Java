Multiplying input integer by 2 and displaying the result

Hey there! This problem might be a bit unpredictable, but give it a go and let us know how you do!
Write a program in Java that reads an integer value, multiplies it by 2 and then prints the result. Your program should be capable of reading the integer value from the console and then printing the double of that number. For example, if 4 is the input number, your program should print 8.
Sample Input:
4

Sample Output:
8


Sample Input:
5

Sample Output:
10



Memory limit: 256 MB
Time limit: 5 seconds
