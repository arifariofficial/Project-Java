Adding comments to a 2D distance calculation program

Hey there! This problem might be a bit unpredictable, but give it a go and let us know how you do!
You have encountered a program that calculates the distance between two points on a 2D surface, but it is completely uncommented and therefore hard to understand. Your task is to add comments to the program to improve its readability. The first line of the input contains two integers, x1 and y1 which is the first point. The second line of the input contains two more integers, x2 and y2 which is the second point. Output the program with the added comments.
Sample Input:
1 2
3 4

Sample Output:
// Taking inputs for first point
int x1 = 1;
int y1 = 2;

// Taking inputs for second point
int x2 = 3;
int y2 = 4;

// Calculating distance
double distance = Math.sqrt((x2-x1)*(x2-x1) + (y2-y1)*(y2-y1));

// Printing the distance
System.out.println(distance);
