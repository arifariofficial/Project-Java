Converting decimal numbers into binary equivalents

Hey there! This problem might be a bit unpredictable, but give it a go and let us know how you do!
Consider that you are given a decimal number. Your task is to convert the decimal number into its binary equivalent and print it. The provided decimal number will be of type double and will have up to two decimal places.
Sample Input:
13.0

Sample Output:
1101


Sample Input:
6.75

Sample Output:
110.11



Memory limit: 256 MB
Time limit: 5 seconds
